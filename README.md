# Forge Dox
Simple theme for Forge.

## Example Config
```yml
Theme: forge-dox
Port: 8000

ThemeParams:
  SiteName: Rendr.js
  SiteLogo: /logo.png
  Menu:
    Main:
      - Name: Introduction
        Url: /
      - Name: Docs
        Url: /docs/getting-started
    Docs:
      - Name: Getting Started
        Url: /docs/getting-started
      - Name: Components
        Url: /docs/components
```
